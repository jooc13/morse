## Accessibility Features

1. **Keyboard Navigation**
   - Tab through accordions
   - Enter/Space to expand/collapse
   - Arrow keys for list navigation

2. **ARIA Labels**
   - `aria-expanded` on accordions
   - `role="region"` for workout sections
   - `aria-label` for icon-only buttons

3. **Focus Indicators**
   - 2px solid outline on focus
   - Visible in all states
   - High contrast ratio (4.5:1 minimum)

4. **Screen Reader Support**
   - Semantic HTML structure
   - Descriptive button text
   - Status announcements for dynamic content

## Interaction Patterns

### Hover States
- Border color changes from `#e8e8e8` to `#606060`
- Subtle shadow appears (4px blur, 8% opacity)
- 2px upward lift on cards
- Cursor changes to pointer on clickable areas

### Click/Tap Feedback
- Scale down to 0.98 on press
- Spring back to 1.0 on release
- 150ms duration

### Expand/Collapse
- Smooth height transition (200ms)
- Icon rotates 180deg
- Content fades in with stagger

### Scroll Behavior
- Smooth scrolling enabled
- Sticky header during scroll
- Load more triggers at 80% scroll
